# Sergei Prilepko - Resume

Резюме на React + TypeScript + Vite

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой на GitHub Pages

1. Собери проект: `npm run build`
2. В настройках репозитория GitHub включи GitHub Pages
3. Выбери папку `dist` как источник
4. Или используй GitHub Actions для автоматического деплоя
